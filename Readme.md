# Requirement

#### Develop a service to get characteristics of given number 
 * Verify :
    * is number prime
    * is number a perfect square
    * is number part of fibonacci series.
    
 * Input : number N(1<=N<=100)
 * Output :
```
    {
      "isPrime" : false,
      "isPerfectSquare" : true,
      "isFibonacci" : true
    }
```
